# Housing Price Prediction Using Machine Learning

## Group Information
Group Q

## Project Overview
This project applies machine learning techniques to predict housing prices based on housing characteristics such as square footage, number of bedrooms, number of bathrooms, neighborhood, and year built.

## Dataset
- housing_price_dataset.csv (original dataset)
- preprocessed_housing_data.csv (processed dataset)

## Machine Learning Models
- Linear Regression
- Decision Tree
- Random Forest
- k-Nearest Neighbors (kNN)
- Gradient Boosting

## Evaluation Metrics
- R² Score
- RMSE
